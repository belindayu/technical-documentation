About HTML

HTML = Hypertext Markup Language

  • It's the skeleton of a person! The structure, contents, etc.

Adding an image

Here's the format:

<img src="[link to image]" altsrc="[alternative text]">
  • No closing tag
  • altsrc = the text that appears when image doesn't load and what screen readers see

About CSS

CSS = Cascading Style Sheets

  • It's the clothes and accessories on a person! The outside appearance, styling, etc.
  • Displays text and other content written in HTML.
  • Case sensitive.

You can control:

  • Color
  • Fonts
  • Positioning
  • Spacing
  • Sizing
  • Decorations
  • Transitions

Choosing a color

The fun part! The choice of color can draw attention to content, evoke emotions, or create visual harmony.

Complementary Colors

  • Opposite of each other on the color wheel

Tertiary Colors

  • Primary color and one of its secondary color neighbors

Adjust the Hue of a Color

hsl(___, ___, ___)
  • Hue: angle of color on the circle (0-360)
  • Saturation: amount of gray (0% - almost completely gray to 100% - fully saturated/normal)
  • Lightness: amount of black or white (0% - black to 50% - normal to 100% - white)

Adjust the Tone of a Color

  • Tone: adding gray / tinting and shading
  • hue + white = tint
  • hue + black = shade

Create a Gradual CSS Linear Gradient

background: linear-gradient (gradient_direction, color 1, color 2, ...);
  • Example:
background: linear-gradient (90deg, red, yellow, rgb(204, 204, 255);

Adding animation

This is also the fun part! CSS allows animation of HTML elements without using JavaScript or Flash!

#anim {
animation_name: colorful;
animation_duration: 3s;
}

Next, we need to specify some keyframes for the animation. Keyframes hold what styles the element will have at certain times.

@keyframes colorful {
0% { // start of animation
background-color: blue;
}
100% { //end of animation
background-color: yellow;
}
}

About JS

JS = JavaScript

  • It's the functionality of a person! The muscles, organs, etc.

Let vs. Var Variables

Let

  • block scoped
  • can only be accessed within the block it was declared

Var

  • function scoped
  • can be accessed globally, even if declared within a block
